Инструкция администратора
Все манипуляции с объектами производяться пользователем с правами администратора.

1. Добавление объекта в приложение
1.1 ПКМ -> меню "Добавить объект" -> форма добавления объектов
1.2 	Левая область - присвоение структурных параметров: название, жанр, режим, цена
	Средняя область - добавление облажки
	Правая область - добавление описания 
1.3 Присвоить структурные параметры объекта
1.3.1 Присвоение имени - ограничения: в имени не должно быть спец. символов
1.4 Выбрать обложку
1.4.1 Формат картинки - jpg, созранение картинка в папку "Загрузки" для дальнейшего инпользования
1.5 Описание

2. Удаление объекта из приложения
2.1 ПКМ -> меню "Удалить объект" -> форма удаления объектов
2.2 Из вырадающего списка выбрать название объекта для удаления
2.3 Для подтверждения выбора - название объекта в TextBox
2.4 Кнопка "Удалить"
